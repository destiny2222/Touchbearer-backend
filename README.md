# Torch Bearer Backend

A Node.js backend application for the Torch Bearer system.

## Features

- Authentication and authorization
- Student management
- Staff management
- Academic management (classes, subjects, assignments, exams)
- Financial management (fees, payments, expenses)
- Analytics and reporting
- And more...

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the application:
   ```
   npm start
   ```

## API Routes

The application includes various API routes for managing different aspects of the system:
- Authentication (`/auth`)
- Students (`/students`)
- Staff (`/staff`)
- Classes (`/classes`)
- Assignments (`/assignments`)
- And many more...
